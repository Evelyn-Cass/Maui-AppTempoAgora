# 🌤️ Maui-AppTempoAgora

A cross-platform mobile application for checking current weather conditions, built with .NET MAUI. Shows users real-time weather information for their location.

## 📱 About The Project

Maui-AppTempoAgora is a simple and practical tool for checking the weather right now. The app displays current temperature, weather conditions, and other relevant information.

This project was created for study purposes and to demonstrate the capabilities of .NET MAUI for cross-platform mobile and desktop development.

- Display current weather conditions
